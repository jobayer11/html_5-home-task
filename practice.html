picture eleement

Resize the browser to see different versions of the picture loading at different viewpoint sizes.The browser looks fir the first source element where the media query matches the user's current viewpoint width, and fetches the image specified in the srcset attribute

The img element is requried as the last child tag of the picture declaration block. The img element is used to provide backward compatibility for browsers that do not support the picture element, or if none of the source tags matched.

Note:The picture element is not supported in IE12 and earlier or Safari 9.0 and earlier
